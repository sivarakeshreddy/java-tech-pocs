# Siva Rakesh Reddy — Portfolio

A single-page React site showcasing Siva Rakesh Reddy’s career highlights, projects, and contact details. Built with Vite for fast local runs.

## Run locally

```bash
npm install
npm run dev
```

Then open the printed local URL (usually `http://localhost:5173`). The résumé PDF lives in `public/SivaRakesh_Resume.pdf` and is linked from the UI.

## Customizing

- Update headline or summary in `src/App.jsx` under the `profile` object.
- Add your portfolio URL by setting `portfolioLink` in `src/App.jsx`.
- Skills, experience, and project outcomes are defined as arrays in `src/App.jsx` for quick edits.
