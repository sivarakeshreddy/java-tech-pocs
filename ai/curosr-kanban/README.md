# Kanban MVP

A minimal Kanban-style project management web app.

## Setup

Requires Node.js 18+ and npm.

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm run test
```

## E2E (Playwright)

```bash
npx playwright install
npx playwright test
```
