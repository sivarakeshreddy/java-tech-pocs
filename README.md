# java-tech-pocs

Collection of technology proof-of-concepts organized by domain.

## Structure

- `frontend/`: Frontend and UI POCs
  - `SITE`: Resume portfolio website (React + Vite)
  - `copilot-kanban`: Kanban POC
  - `cursor-kanban`: Kanban POC
  - `kanban`: Kanban POC
- `java/`: Java-specific POCs
- `ai/`: AI-only assets (agents, prompts, workflows)

## Notes

- Each POC keeps its own README and dependencies.
- Keep nested Git repositories out of subfolders unless using submodules intentionally.
